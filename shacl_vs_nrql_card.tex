\vspace{5pt}
{\large\color{muted} A constraint that says ``every \texttt{Person} must have a known \texttt{firstName}''\par}

\vspace{12pt}

\begin{tcbraster}[raster columns=2, raster column skip=8mm, raster equal height=rows,
  colframe=black!12, colback=panelbg, boxrule=0.8pt, arc=4mm,
  fonttitle=\large\bfseries\color{white}, left=4mm, right=4mm, top=3mm, bottom=3mm]

\begin{tcolorbox}[title={SHACL Core shape \ (W3C, 2017)}, colbacktitle=shaclblue]
\begin{lstlisting}[language=turtle]
ex:ParentShape
  a sh:NodeShape ;
  sh:targetClass ex:Person ;
  sh:property [
    sh:path ex:firstName ;
    sh:minCount 1
  ] .
\end{lstlisting}
\end{tcolorbox}

\begin{tcolorbox}[title={RacerPro nRQL \ (already possible in 2005)}, colbacktitle=nrqlgreen]
\begin{lstlisting}[language=nrql]
(retrieve (?x)
  (and (?x Person)
       (neg
         (project-to (?x)
           (?x ?y firstName)))))
\end{lstlisting}
\end{tcolorbox}

\end{tcbraster}

\vspace{8pt}
{\large\color{ink}
A \textbf{negated projected subquery} $\;=\;$ \texttt{NOT EXISTS} / anti-join \;---\;
nRQL had it \emph{years} before SPARQL\,1.1 (2013) standardized \texttt{FILTER NOT EXISTS}.\par}
