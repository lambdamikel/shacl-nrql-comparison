# RacerPro manuals

The official RacerPro documentation, referenced throughout this repository (the
paper's bibliography, the [nRQL guide](../nrql-guide/), and the
[verification scripts](../verification/)).

- **[RacerPro User's Guide 2.0](RacerPro-Users-Guide-2.0.pdf)** (Racer Systems
  GmbH & Co. KG, 2012) — nRQL is documented in §4.1; the formal nRQL grammar is
  §4.1.9 (pp. 150–153); the data/mirror substrate is pp. 137–143.
- **[RacerPro Reference Manual 1.9](RacerPro-Reference-Manual-1.9.pdf)** (2010).

Also available in the RacerPro source repository:
<https://github.com/lambdamikel/Racer/tree/master/doc>.
